ReactJS app based on the Prudential HTML Template

## How to run

1. git clone git@github.com:soubhikchatterjee/prudential-react-app.git
2. cd prudential-react-app
3. npm i
4. npm start

## How to build

1. git clone git@github.com:soubhikchatterjee/prudential-react-app.git
2. cd prudential-react-app
3. npm i
4. npm run build
